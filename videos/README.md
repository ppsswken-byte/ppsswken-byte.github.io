# โฟลเดอร์วิดีโอสำหรับจอ TV

อัปโหลดไฟล์วิดีโอ 2 ไฟล์ไว้ที่โฟลเดอร์นี้ (ชื่อต้องตรงตามนี้):

| ไฟล์ | ใช้ช่วง |
|---|---|
| `lunch.mp4` | ช่วงลันช์ — วันธรรมดา 10:45–15:01 / วันหยุด 10:45–16:01 (ค่าเริ่มต้น) |
| `dinner.mp4` | ช่วงดินเนอร์ — ทุกวัน 16:45–23:01 (ค่าเริ่มต้น) |

- ฟอร์แมต: **MP4 (H.264)** ขนาดไม่เกิน **100 MB ต่อไฟล์** (ข้อจำกัดของ GitHub)
- ถ้าไฟล์ใหญ่เกิน ให้บีบอัดด้วย [HandBrake](https://handbrake.fr) preset "Fast 1080p30"
- วิดีโอเล่นแบบไม่มีเสียง (muted) วนลูปอัตโนมัติ
- เวลาสลับ / บังคับโหมด / วันหยุดนักขัตฤกษ์ ควบคุมได้จากหน้า **staff-dashboard.html** ในกล่อง "TV ディスプレイ映像"
- ช่วงเวลานอกตาราง (เช่น ปิดร้าน หรือช่วงพักบ่าย) จอจะแสดงโลโก้ร้านแทน
- ถ้าอยากใช้หลายไฟล์ต่อช่วง แก้รายชื่อไฟล์ได้ที่ตัวแปร `LUNCH_VIDEOS` / `DINNER_VIDEOS` ใน `tv-display.html`
